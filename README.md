🎯 What is CoderBuddy?
CoderBuddy is an intelligent AI agent that transforms your project descriptions into complete, production-ready web applications. Simply describe what you want to build, and watch as CoderBuddy generates HTML, CSS, JavaScript, and full project structures in seconds.
✨ Perfect For:

🎨 Landing pages & portfolios
📝 Quiz & survey applications
🎮 Interactive web games
📊 Data visualization dashboards
🛍️ E-commerce templates


🌟 Features
FeatureDescription🤖 AI-PoweredLeverages Groq's lightning-fast LLM API for instant code generation🎨 Full-StackGenerates complete HTML/CSS/JS projects with proper structure🔧 Smart AgentIntelligent planning and iterative code improvement⚡ Blazing FastGet your app in seconds, not hours📦 Zero ConfigWorks out of the box with minimal setup🎯 Production ReadyClean, documented, and deployable code

🚀 Quick Start
Prerequisites
bashPython 3.8+
Groq API Key
Installation

Clone the repository

bashgit clone https://github.com/tarunmehrda/CoderBuddy-AI-Code-Generation-Assistant-FINAL.git
cd CoderBuddy-AI-Code-Generation-Assistant-FINAL

Install dependencies

bashpip install -r requirements.txt

Set up your Groq API key

bashexport GROQ_API_KEY="your_groq_api_key_here"

Run CoderBuddy

bashpython main.py

💡 Usage
python# Simply describe your project
"Create a quiz application with multiple-choice questions, 
score tracking, and a beautiful gradient UI"

# ✅ Responsive CSS styling
